************************************* Explication de compilation ************************************


Afin de compiler notre IA, il faut faire un "make chaine" dans le dossier src puis aller dans le dossier test/ pour pouvoir faire les tests.


*********************************** Lancement des différents test ***********************************


Le dossier test contient les différents scripts tests que nous avons éffectués :


- Pour tester notre IA contre les bots fournis (qui se trouvent dans le dossier client/java), il faut lancer le script "test.sh" avec en premier paramètre le nom du bot adverse et en deuxieme paramètre le numéro de la carte (de 0 à 100). Nous sommes le joueur 1 (rouge).

- Pour tester notre IA contre des bots d'autres groupes de projet, on procède de la même manière que précedemment en lançant le sccript "testjoueurs.sh" et en choisissant le nom du bot adverse parmis ceux presents dans le dossier ./autresbot. Nous sommes le joueur 1 (rouge).

- Pour tester la partie expansion et attaque de l'IA, lancer le script "test_exp_att.h". Nous sommes le joueur 1 (rouge).

- Pour tester la défense, il faut lancer le script "test_defense.sh". Nous sommes le joueur 1.

- Pour finir, si l'on veut tester notre IA sur l'ensemble des bots, il suffit de lancer le script "test_all.sh" et de suivre les instructions. Après la fin de l'execution du script, les resultats pour chaque map sont écrits dans un fichier "score.txt".
